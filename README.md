# EASY RECIPEDIA

Easy Recipedia is a Single Page Application(SPA) that allows users to create and view recipes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'easy_recipedia'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install easy_recipedia


## Requirements
